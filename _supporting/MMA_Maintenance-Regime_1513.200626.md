# MMA Maintenance Regime — Dynamic Baseline

## 1. Objective

Maintain a mobile, flexible, strong, resilient, dynamic baseline for MMA.

Not fight camp. Not conditioning. Not another heavy session.

## 2. Daily baseline — 12–15 min

| Area           | Exercise                                | Dose                         |
| -------------- | --------------------------------------- | ---------------------------- |
| Neck           | Rotations                               | 3–5 each direction           |
| Neck           | Side bends                              | 3–5 each side                |
| Shoulders      | Shoulder circles                        | 10 forward / 10 backward     |
| Shoulders      | Arm swings                              | 10–20 total                  |
| Thoracic spine | Cat-cow                                 | 6–10 reps                    |
| Thoracic spine | Open book rotations                     | 5 each side                  |
| Wrists         | Wrist circles                           | 10 each direction            |
| Wrists         | Palm-down wrist rocks                   | 10–15 reps                   |
| Wrists         | Palm-up wrist rocks                     | 10–15 reps                   |
| Hands          | Finger spreads / extensions             | 10–20 reps                   |
| Hips           | Hip circles                             | 5 each direction / each side |
| Hips           | 90/90 switches                          | 6–10 total                   |
| Hips           | Leg swings front/back                   | 10 each leg                  |
| Hips           | Leg swings side/side                    | 10 each leg                  |
| Knees          | Knee circles                            | 5 each direction             |
| Ankles         | Knee-over-toe rocks                     | 10 each side                 |
| Feet / toes    | Toe spreads                             | 10–20 reps                   |
| Feet / toes    | Big-toe lift                            | 10 each foot                 |
| Feet / toes    | Other-toes lift                         | 10 each foot                 |
| Feet           | Short-foot / arch activation            | 5–10 x 3-sec holds each foot |
| Squat          | Relaxed squat hold                      | 30–60 sec                    |
| Adductors      | Side-to-side adductor shift             | 5–8 each side                |
| Dynamic        | Bounce / stance switch / light footwork | 60–90 sec                    |

## 3. Wrist / hand block

Daily or before MMA.

| Exercise                         | Dose                |
| -------------------------------- | ------------------- |
| Wrist circles                    | 10 each direction   |
| Palm-down wrist rocks            | 10–15 reps          |
| Palm-up wrist rocks              | 10–15 reps          |
| Fist rocks                       | 5–10 reps           |
| Finger spreads                   | 10–20 reps          |
| Finger extensions                | 10–20 reps          |
| Light fist plank / knuckle plank | 10–20 sec, optional |

2–3x/week.

| Exercise                 | Dose                               |
| ------------------------ | ---------------------------------- |
| Farmer carries           | 2–4 carries of 20–40 sec           |
| Towel hangs / towel rows | 2–3 sets of 10–30 sec or 6–10 reps |
| Wrist curls              | 2 sets of 10–15                    |
| Reverse wrist curls      | 2 sets of 10–15                    |
| Pronation / supination   | 2 sets of 8–12 each side           |
| Grip squeezes            | 2 sets, not to failure             |

Rule: prepare, do not inflame.

## 5. Pre-MMA activation — 8–12 min

Use before training.

| Exercise                                              | Dose        |
| ----------------------------------------------------- | ----------- |
| Light bounce / skipping / shadow footwork             | 2 min       |
| Wrist / hand prep                                     | 1–2 min     |
| Foot / ankle prep                                     | 1–2 min     |
| Hip openers                                           | 5 each side |
| Leg swings front/back                                 | 10 each leg |
| Leg swings side/side                                  | 10 each leg |
| Glute bridges                                         | 10–15 reps  |
| Dead bug / plank                                      | 30–45 sec   |
| Scap push-ups / band pull-aparts                      | 10–15 reps  |
| Technical ramp: shadowboxing, sprawls, checks, pivots | 2–3 min     |

Finish warm and sharp, not tired.

## 6. Post-MMA reset — 8–15 min

Use after training.

| Exercise                | Dose                |
| ----------------------- | ------------------- |
| Slow nasal breathing    | 2 min               |
| Hip flexor stretch      | 30–60 sec each side |
| Hamstring stretch       | 30–60 sec each side |
| Adductor stretch        | 30–60 sec each side |
| Calf stretch            | 30–60 sec each side |
| Foot release            | 30–60 sec each foot |
| Wrist / forearm release | 30–60 sec each side |
| Thoracic rotation       | 5 each side         |
| Pec / shoulder stretch  | 30–60 sec each side |

Reset only. Not another workout.

| Carry   | Farmer carry               | 2–4 carries of 20–40 sec |

Session B.

| Pattern      | Exercise                        | Dose                       |
| ------------ | ------------------------------- | -------------------------- |
| Lunge        | Reverse lunge / walking lunge   | 2–3 sets of 6–10 each side |
| Push         | Overhead / incline push         | 2–3 sets of 6–10           |
| Pull         | Row / pull-up                   | 2–3 sets of 6–12           |
| Hip          | Bridge / hip thrust             | 2–3 sets of 8–12           |

| Wrist / grip | Carries / towel work            | 2–3 sets                   |

Rule: moderate effort. Leave reps in reserve.

## 8. Readiness check

| State | Action                            |
| ----- | --------------------------------- |
| Green | Train normally                    |
| Amber | Technical only / reduce intensity |
| Red   | Recovery only                     |

Red signs:

* sharp pain
* unstable joint
* worsening pain during warm-up
* unusual fatigue
