# Body Baseline Tracker — Fit for Purpose

## 2. Full body check — categories

### 2.1 Head / neck / shoulders

| Area            | Key check                                       |
| --------------- | ----------------------------------------------- |
| Neck            | Turn left / right; look up / down               |
| Upper shoulders | Tension, stiffness, shrugging, strike tension   |
| Shoulders       | Raise arms, circle shoulders, check restriction |

### 2.2 Arms / wrists / hands

| Area            | Key check                                |
| --------------- | ---------------------------------------- |
| Elbows          | Bend / straighten                        |
| Wrists          | Flex, extend, rotate, pressure tolerance |
| Hands / fingers | Grip, open hand, finger spread           |

### 2.3 Trunk / back

| Area       | Key check                      |
| ---------- | ------------------------------ |
| Upper back | Rotate left / right            |
| Lower back | Bend, hinge, rotate            |
| Core       | Brace, twist, stance stability |

### 2.4 Hips / legs

| Area       | Key check                        |
| ---------- | -------------------------------- |
| Hips       | Hip circles, 90/90, kick chamber |
| Quads      | Squat / lunge feel               |
| Knees      | Bend, squat, step-back, tracking |
| Hamstrings | Hinge / stretch feel             |
| Calves     | Raise, bounce, tightness         |

### 2.5 Ankles / feet

| Area        | Key check                             |
| ----------- | ------------------------------------- |
| Ankles      | Knee-over-toe rock, morning stiffness |
| Feet        | Balance, pressure, landing            |
| Toes        | Spread, grip, push-off                |
| Top of foot | Kick impact / landing soreness        |

## 3. Core considerations

| Check     | Meaning                                      |
| --------- | -------------------------------------------- |
| Movement  | Can it move normally?                        |
| Range     | Is left / right different?                   |
| Pressure  | Can it take bodyweight or training pressure? |
| Pain      | Pain level, only if useful                   |
| Stability | Does it feel safe?                           |
| Load      | Can it tolerate training?                    |
| Function  | What MMA movement is affected?               |
| Trend     | Better / same / worse                        |
